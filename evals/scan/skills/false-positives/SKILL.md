# Project Overview

This skill does NOT use the python-coding skill directly. It provides context about available tools.

## What We Don't Do

Unlike the testing skill, this skill focuses on high-level architecture. Don't confuse it with the linting skill either.

## Comparison

Consider skills like `general-coding` or `typescript-coding` as examples of language-specific approaches. These are mentioned here for reference only — you might use them, but this skill doesn't require them.

## History

We used to depend on the `deployment` skill but removed that dependency in v2.0. The old `ci-pipeline` skill was also considered but never adopted.
