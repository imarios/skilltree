# Code Review

When reviewing code, follow this process:

1. **LOAD** `linting` skill for style and lint checks
2. Use the testing skill for coverage validation
3. Load the general-coding skill for cross-language principles
4. Apply the `typescript-coding` skill when reviewing TypeScript files

Always check that tests pass before approving.
