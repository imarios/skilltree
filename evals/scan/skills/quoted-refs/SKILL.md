# Onboarding Checklist

## Day 1

1. Clone the repo and run `skilltree install`
2. Read the <general-coding> skill to understand our coding standards
3. Review the "typescript-coding" skill for our TypeScript conventions
4. Skim the 'python-coding' skill if you'll work on backend services

## Week 1

5. Shadow a code review using the `code-review` skill guidelines
6. Complete the testing skill's recommended exercises
7. Set up your editor with the linting skill's configuration
